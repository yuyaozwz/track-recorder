# 基于Android的行踪记录器

## 介绍

该系统是一个基于Android的行政记录器，目的是对用户的定位信息进行监控。
该系统分为 用户端 和 管理端。

用户端是定位数据的生产者，每隔1秒将当前的定位信息发送至服务器。
管理端是定位数据的消费者，每隔1秒将定位信息从服务器上拉下来。

## 安装教程

1. 使用 Android Studio，导入本项目
2. 启动第三方模拟器（如雷电模拟器）
3. 运行项目

## 使用说明

启动项目必须开启手机定位权限，否则无法定位。
